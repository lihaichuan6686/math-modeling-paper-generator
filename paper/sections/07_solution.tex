\section{Solution Process}

The solution is generated by \texttt{src/demo\_v1.py}. The script:

\begin{enumerate}
  \item defines synthetic demand and supplier data;
  \item computes supplier scores;
  \item builds an eight-week greedy order plan;
  \item saves the result table as CSV and LaTeX;
  \item draws the inventory validation figure;
  \item writes a run summary under \texttt{runs/v1-demo/}.
\end{enumerate}

All generated artifact paths are stable so that the paper and review files can reference them directly.
