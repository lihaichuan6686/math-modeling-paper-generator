\section{Data Processing}

The demo data are synthetic and are stored in the problem statement. Demand is given for eight weeks. Supplier information includes capacity, unit price, transport loss rate, and reliability.

For a real contest problem, this section would include missing-value checks, abnormal supplier behavior, unit conversion, and feature construction. For v1.0, the data are already clean, so the processing step focuses on converting the supplier table into evaluation scores and planning parameters.
