# Review Checklist

## 合规与用途

- [ ] 本次使用场景属于学习、复盘、课程研究或授权研究。
- [ ] 没有要求隐藏 AI 参与或绕过竞赛规则。
- [ ] 没有伪造数据、伪造引用或伪造实验。

## 题面理解

- [ ] 每个子问题都被覆盖。
- [ ] 输入、输出和约束没有遗漏。
- [ ] 没有把题面之外的条件当成已知事实。

## 模型

- [ ] 变量、参数和单位定义清楚。
- [ ] 假设有合理解释。
- [ ] 目标函数和约束可以对应到题面。
- [ ] 推导没有明显跳步。
- [ ] 模型局限被明确说明。

## 代码与实验

- [ ] 代码可以从头运行。
- [ ] 随机过程设置了种子或说明了随机性。
- [ ] 图表由代码生成。
- [ ] 关键结果能复现。
- [ ] 没有把测试输出误写成正式结论。

## 图表

- [ ] 每张图表都有编号和标题。
- [ ] 坐标轴、单位和图例清楚。
- [ ] 图表在正文中被解释。
- [ ] 表格数值和正文结论一致。

## 写作

- [ ] 摘要包含方法、结果和结论。
- [ ] 正文结构符合数模论文习惯。
- [ ] 结论没有超过模型和数据支持范围。
- [ ] 语言简洁，避免空泛套话。

## LaTeX

- [ ] PDF 可以成功编译。
- [ ] 没有缺图、缺表、未定义引用。
- [ ] 公式、图表和章节编号正常。
- [ ] 版面没有明显溢出或重叠。
