## Known Tooling Risk

`scripts/check-v1.6-layout.ps1` expects the project Python environment to contain `pypdf`.

If the wrapper reports missing `pypdf`, run:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\setup.ps1
```

Then rerun the layout check.

The final checker reports this as a failed environment gate, and the layout checker reports dependency failures as a `Dependency Gate` item without crashing, so the user still gets readable `reviews/final-v16-check.md` and `reviews/layout-v16-check.md` repair paths.
