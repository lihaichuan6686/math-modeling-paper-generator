# Sample Run Packages

Purpose: give Claude Code one short doorway into the strongest route-specific reference package currently available in the repository.

This layer is lighter than a full demo paper and more practical than reading every note separately.

Each package points Claude to:

- the filled run files;
- the section-level writing examples;
- the paper-assembly map;
- the bundle claim/caption controls;
- the review case example.

## Included Packages

1. `evaluation-to-planning-demo/`
2. `rail-timetable-demo/`

Each package contains:

- `package-map.md`

## How To Use

When Claude is about to start a real v1.2 run for a route family:

1. identify the nearest route;
2. read that package first;
3. then read the linked run files, section examples, bundle maps, and review case;
4. only then start drafting or revising.
