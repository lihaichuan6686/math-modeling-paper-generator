# v1.6 Feedback Triage Matrix

Purpose: convert v1.6 user-side feedback into concrete repository repairs.

## Repair Layer Rule

| If the failure is... | Repair this layer first |
|---|---|
| Claude does not create required run files | `scripts/new-run.ps1`, `scripts/start-current.ps1`, skill entry |
| Claude ignores page targets | `prompts/16_launch_v1_6.md`, `section-rhythm-budget.md`, layout checker |
| Abstract is short or spills | `knowledge/latex/v1-6-layout-rhythm-rules.md`, PDF checker |
| Concept figure is generic | `knowledge/visuals/v1-6-nature-style-figure-rules.md`, figure spec template |
| Figure labels unreadable or boxed | figure-generation code, PDF visual gate, font rules |
| Table overflows | `paper/templates/table-safe-snippets.tex`, LaTeX table rules, layout checker |
| PDF is long because of blank pages | section rhythm rules, symbol compaction rules, layout checker |
| Review misses obvious defects | `knowledge/quality/v1-6-layout-hard-gates.md`, review scaffold |
| Checker misses a repeated defect | `scripts/check-v1.6-layout.py` |

## Triage Table

| Feedback bucket | Evidence to collect | Repair output |
|---|---|---|
| Page rhythm | page count, page density list, sparse-page screenshots | updated section rhythm rules or checker thresholds |
| Abstract | page 1 screenshot, extracted char count, abstract source | updated abstract target or prompt instruction |
| Concept figure | screenshot, figure-style-spec.md, source script | updated visual rule or generation template |
| Tables | log overfull lines, screenshot, table source | safe-table snippet or checker upgrade |
| Figures | screenshot, source script, font settings | figure style rule or source-code helper |
| Self-review | review-current.md, layout-v16-check.md | review scaffold/gate repair |

## Stop Rule

Do not add broad new features before classifying the failure. Every v1.6 test failure should become one of:

- prompt repair;
- template repair;
- checker repair;
- knowledge/rule repair;
- source-generation helper repair;
- deferred item with reason.
