# Run File Examples

Purpose: provide realistic example run files that Claude can imitate when filling `runs/current/` for a real v1.2 paper attempt.

These are not the only valid shapes. They are meant to show what a non-placeholder run record looks like after route selection is already clear.

## Included Families

1. `evaluation-to-planning-demo/`
2. `rail-timetable-demo/`
3. `forecast-to-decision-demo/`
4. `geometry-spatial-demo/`
5. `dynamic-scheduling-demo/`
6. `classification-recognition-demo/`
7. `monitoring-route-e-demo/`

Each example folder contains:

- `problem-profile.md`
- `problem-analysis.md`
- `route-decision.md`
- `model-plan.md`
- `section-budget.md`
- `writing-style-plan.md`
- `verification-plan.md`
- `figure-plan.md`
- `methodology-checklist.md`
- `literature-notes.md` when the route example includes source/citation discipline
- `online-consensus-notes.md` when the route example includes public-signal discipline
- `artifact-ledger.md`
