# Verification Plan

Run: example-monitoring-route-e

## Main Checks

1. verify that the paper contains an explicit monitoring rule, policy table, or intervention scheme;
2. verify that the rule is linked back to the diagnosed pattern or trigger;
3. verify that intervention or long-horizon consequence evidence exists;
4. verify that abstract and conclusion state the policy scope rather than only the detected pattern.

## Required Artifacts

- diagnostic artifact
- monitoring/policy artifact
- consequence comparison artifact
- recommendation summary artifact

## Review Warnings

- fail if the paper stops at diagnostics
- fail if the policy or rule has no traceable trigger
- warn if the conclusion implies broad management effectiveness beyond the tested horizon
