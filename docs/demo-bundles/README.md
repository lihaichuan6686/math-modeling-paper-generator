# Demo Bundles

Purpose: provide the closest thing to a reusable end-to-end sample run without pretending that a full real contest paper has already been generated for every route.

Each bundle ties together four layers:

- filled run-file examples;
- section-level writing examples;
- paper-assembly guidance;
- final handoff checks.

This is the bridge for Claude when it already understands the rules, but still needs to see how one route becomes a coherent research draft package.

## Included Bundles

1. `evaluation-to-planning-demo/`
2. `rail-timetable-demo/`

Each bundle contains:

- `bundle-map.md`
- `claim-map.md`
