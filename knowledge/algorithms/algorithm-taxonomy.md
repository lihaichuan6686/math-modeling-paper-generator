# Algorithm Taxonomy

这是数模生成器的算法知识骨架。后续每类算法都会补充适用条件、输入输出、常见公式、Python 实现建议、LaTeX 表达方式和质量检查点。

## 优化与规划

- 线性规划、整数规划、非线性规划
- 多目标规划
- 动态规划
- 启发式优化：遗传算法、模拟退火、蚁群、粒子群

## 评价与决策

- 层次分析法
- 熵权法
- TOPSIS
- 灰色关联分析
- 模糊综合评价
- 主成分分析

## 预测与统计

- 回归分析
- 时间序列
- 灰色预测
- 马尔可夫模型
- 机器学习预测
- 不确定性分析

## 图与网络

- 最短路、最小生成树、最大流
- 社区发现
- 网络中心性
- 匹配与分配

## 仿真与机理

- 蒙特卡罗
- 元胞自动机
- 微分方程
- 排队论
- 多智能体仿真

## 数据处理与拟合

- 插值
- 曲线拟合
- 聚类
- 降维
- 异常检测

## 论文生成要求

每次选择算法时必须回答：

1. 为什么这个算法适合题目？
2. 输入数据是否满足算法前提？
3. 参数如何确定？
4. 结果如何验证？
5. 算法局限是什么？
6. 是否有更简单、可解释的替代方案？
