## Maintenance Rule

When new work appears, decide which bucket it belongs to:

| Bucket | Examples |
|---|---|
| Route | route index, paper-family matrix |
| Flow | generation loop, prompt flow map |
| Evidence | visual/table/equation family indexes |
| Review | completion gate, comparison index |
| Community | soft rules, source-quality rubric, online-consensus notes |
| Tracking | learning-status, continuation-state |

If a new item does not fit one of these buckets, the knowledge stack is probably missing a layer.
