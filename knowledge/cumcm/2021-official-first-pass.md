## A 题：“FAST” 主动反射面调节模型

- 样本：`2021A028`
- 题型：空间几何、工程物理机理、约束调节、误差评价
- 核心问题：
  - 给定观测目标方位后，将 FAST 主动反射面从球面调节为近似抛物面。
  - 求主索节点、促动器伸缩量、反射面板调整后的几何关系。
  - 建立馈源舱接收比、照明区域、有效接收面积等评价指标。
- 方法信号：
  - 坐标变换
  - 球面与抛物面方程
  - 主索节点约束
  - 空间几何拟合
  - 面板反射路径分析
  - 电磁波接收比评价
- 摘要写法特征：
  - 先交代工程目标和调节对象，再按问题逐段说明模型。
  - 摘要中直接给出关键方程、关键节点、促动器变化范围和接收比提升。
  - 公式、指标、约束和结果同时出现，形成可复现的摘要。
- 对生成器的启发：
  - 几何工程题不能只给优化算法，必须先定义坐标系、面方程、节点编号、空间约束和误差指标。
  - 对每一个物理量都要说明几何意义和工程限制，例如伸缩范围、边界节点、照明区域。
  - 图应优先生成三类：坐标系示意图、调节前后曲面对比图、误差/节点位移热力图。

## B 题：C4 烯烃制备分析与试验设计

- 样本：`2021B007`
- 题型：化工机理、数据分析、统计建模、优化与试验设计
- 核心问题：
  - 分析乙醇转化率、C4 烯烃选择性和温度、催化剂组合之间的关系。
  - 评估温度对目标产物收率的影响。
  - 在不同约束下寻找最优催化剂组合与温度。
  - 设计补充试验方案。
- 方法信号：
  - 多元回归
  - 偏最小二乘回归
  - 相关性分析
  - 方差分析
  - 非线性拟合
  - 优化求解
  - 均匀试验设计
- 摘要写法特征：
  - 逐问输出明确的化学指标：乙醇转化率、C4 烯烃选择性、C4 烯烃收率。
  - 每个结论都绑定温度范围、催化剂组合或显著性结果。
  - 最后把还需要什么实验作为模型闭环，而不是只给已有数据的拟合。
- 对生成器的启发：
  - 化工/材料题需要“数据探索 -> 统计检验 -> 机理解释 -> 优化方案 -> 试验设计”的完整链条。
  - 摘要中应避免只说建立模型，要写出显著因素、最优条件和实验补充逻辑。
  - 若数据量不大，模型应偏重可解释统计和稳健性检查，而不是过度堆叠黑箱模型。

## C 题：生产企业原材料订购与运输的评价与规划模型

- 样本：`2021C066`
- 题型：供应链评价、采购规划、运输规划、多目标优化
- 核心问题：
  - 从历史供货数据中评价供应商重要性和稳定性。
  - 确定最重要供应商集合。
  - 制定满足企业产能需求的订购与转运方案。
  - 在不同产能目标下评估企业能力边界。
- 方法信号：
  - TOPSIS
  - 评价指标体系
  - 选优筛选
  - 线性规划
  - 遗传算法
  - 多目标规划
  - 灵敏度分析
- 摘要写法特征：
  - 从评价模型进入规划模型，再进入产能上限分析。
  - 结果表达包含供应商编号、转运方案、成本/损耗、产能增量等实际决策量。
  - 不是单纯给排名，而是把排名作为后续采购与运输优化的输入。
- 对生成器的启发：
  - 供应链题的论文应分成两层：评价筛选层和决策优化层。
  - 评价指标必须能进入后续规划，否则容易成为孤立装饰。
  - 输出应包括可执行清单：供应商选择、订购量、承运商分配、成本或损耗变化。

## D 题：连续切割的在线优化模型

- 样本：`2021D017`
- 题型：工业过程优化、在线决策、整数线性规划、异常处理
- 核心问题：
  - 在用户目标长度和尾坯损失之间寻找最优切割方案。
  - 在结晶器异常、报废段出现时实时调整方案。
  - 针对不同目标值输出初始方案与调整后方案。
- 方法信号：
  - 整数线性规划
  - 平均绝对偏差
  - 在线更新
  - 分段情景分析
  - 最优方案调整
- 摘要写法特征：
  - 先证明或说明尾坯损失为零的条件，再进入优化。
  - 用“正常情形 -> 异常情形 -> 不同目标值情形”展开问题。
  - 摘要中给出调整次数、损失长度、偏差范围等可核验指标。
- 对生成器的启发：
  - 在线优化题要区别初始计划和实时调整策略，不能只求一次静态最优。
  - 异常情形需要单独建模：触发条件、可调整变量、保持不变的约束、目标函数变化。
  - 结果图表应包含甘特式切割安排、调整前后损失对比、目标值偏差分布。

## E 题：基于机器学习的中药材类别与产地鉴别

- 样本：`2021E014`
- 题型：机器学习分类、光谱数据处理、特征选择、产地鉴别
- 核心问题：
  - 对近红外与中红外光谱数据进行预处理。
  - 判断不同种类中药材的分类能力。
  - 鉴别中药材产地。
  - 比较不同特征处理和分类器组合的表现。
- 方法信号：
  - Savitzky-Golay 平滑
  - 多元散射校正 MSC
  - 标准正态变换 SNV
  - 主成分分析 PCA
  - 线性判别分析 LDA
  - 支持向量机 SVM
  - 随机森林
  - 特征选择
- 摘要写法特征：
  - 方法链非常清晰：预处理、特征提取、模型训练、分类评价。
  - 以准确率、F1 分数等指标证明模型有效。
  - 对不同任务分别给出最佳方法组合，而不是强行用一个模型解释所有问题。
- 对生成器的启发：
  - 机器学习题必须有“数据划分、预处理、特征选择、模型比较、指标评价”的最小闭环。
  - 分类任务不能只报准确率；应加入混淆矩阵、F1 分数、类别不平衡说明。
  - 光谱/高维数据要优先考虑降维和可解释性，说明主成分或特征波段的意义。

## 横向吸收

### 1. 2021 年的五类能力覆盖

2021 A-E 正好覆盖了国赛生成器必须掌握的五个大类：

- 几何工程：坐标、曲面、物理约束和误差指标。
- 化工材料：统计关系、显著性、机理解释和试验设计。
- 供应链规划：评价筛选、优化分配和产能边界。
- 在线优化：实时调整、异常场景和损失控制。
- 机器学习：预处理、特征工程、分类模型和评价指标。

### 2. 摘要的共性模板

优秀摘要不是背景介绍，而是“逐问压缩版论文”：

```text
问题背景与总体目标
-> 问题一：模型/方法 + 关键结果
-> 问题二：模型/方法 + 关键结果
-> 问题三：模型/方法 + 关键结果
-> 扩展/验证/实验设计
-> 关键词
```

### 3. 对智能体的策略要求

- 先判题型，再选模型链；不要从算法库中随机挑方法。
- 每一问都要形成“变量、目标、约束、求解、结果、验证”的闭环。
- 当题目含有现实行业背景时，必须把业务规则翻译成数学约束。
- 摘要、正文、图表和代码要共享同一套变量名和结果编号。
- 结果应尽量可核验：编号、数值、区间、阈值、准确率、损失、成本、误差。

## 纳入后续知识库的规则

1. 若题面出现空间方位、反射、曲面、节点、几何调节，优先进入“几何工程建模”流程。
2. 若题面出现催化剂、转化率、选择性、收率、温度，优先进入“统计检验 + 化工优化 + 试验设计”流程。
3. 若题面出现供应商、订购、运输、产能、损耗，优先进入“评价筛选 + 线性/整数规划”流程。
4. 若题面出现实时、异常、切割、调度、调整，优先进入“在线优化 + 情景更新”流程。
5. 若题面出现光谱、图像、类别、产地、识别，优先进入“预处理 + 特征工程 + 模型比较”流程。
