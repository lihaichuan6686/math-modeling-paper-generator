## A 题：定日镜场的输出功率优化

- 文件：`A0127.pdf`
- 页数：50
- 题型：新能源工程优化、几何建模、布局规划
- 摘要结构：逐问题写法，围绕镜场坐标、镜面姿态、光学效率、阴影遮挡、截断效率和规划策略展开。
- 关键词：定日镜平均输出热功率优化模型、变步长搜索算法、分区域同心圆规划策略。
- 方法信号：
  - 坐标系建立和坐标转换
  - 反射定律与法向量计算
  - 光线离散化
  - 阴影遮挡判断
  - 变步长搜索
  - 分区域同心圆规划
- 对生成器的启发：
  - 工程几何题要先建立坐标和物理量，再谈优化。
  - 摘要中要给出关键工程指标的数值结果。
  - 布局优化需要解释约束来源和搜索策略。

## B 题：基于主要目标法的测线设计问题

- 文件：`B311.pdf`
- 页数：31
- 题型：几何推导、测线规划、多目标优化
- 摘要结构：开头直接给出关键公式，再按问题推进到推广、递推和多目标规划。
- 关键词：解三角形、类比法、递推法、多目标规划主要目标法。
- 方法信号：
  - 几何分析
  - 正弦定理
  - 覆盖宽度模型
  - 重叠率模型
  - CAD 辅助比较
  - Python/Matlab 计算
  - 递推法
  - 多目标规划主要目标法
- 对生成器的启发：
  - 几何类题的摘要可以直接展示核心公式。
  - 当模型存在多个定义时，应比较并说明选择理由。
  - 多目标问题要说明主目标和约束目标的取舍。

## C 题：某商超蔬菜类商品动态定价与补货决策研究

- 文件：`C050.pdf`
- 页数：41
- 题型：数据分析、预测、定价与补货优化
- 关键词：K-means++ 聚类、优化模型、模拟退火模型、灰色关联分析。
- 方法信号：
  - 描述统计与可视化
  - Spearman 相关分析
  - K-means++ 聚类
  - Pearson 相关分析
  - 线性回归
  - 时间序列预测
  - 利润最大化优化模型
  - 模拟退火
  - 灰色关联分析
- 对生成器的启发：
  - 数据题应先做描述统计、相关性和聚类，再进入预测/优化。
  - 摘要要明确数据指标、预测对象、优化目标和最终收益。
  - 同一题中可组合统计分析、机器学习和优化算法，但每一步要有输入输出关系。

- 文件：`D039.pdf`
- 页数：27
- 题型：生产计划、空间利用率、整数规划、仿真
- 关键词：圈养湖羊、遍历、蒙特卡罗方法、MATLAB。
- 方法信号：
  - 生长周期与批次安排
  - 决策变量简化
  - 非线性整数规划
  - 遍历搜索
  - 蒙特卡洛模拟
  - 损失数据建模

- 文件：`E176.pdf`
- 页数：50
- 题型：水文时间序列、突变检测、采样优化
- 关键词：双对数回归、M-K 检验、小波变换、R/S 分析、傅里叶级数、0-1 规划、微元法。
- 方法信号：

  - 正弦函数拟合
  - 双对数回归
  - 描述性分析
  - 季节性、周期性、突变性分析
  - M-K 检验
  - Fisher 最优分割
  - 小波变换
  - R/S 分析
  - 傅里叶级数预测

- 对生成器的启发：

  - 监测方案题通常需要把预测问题转化为优化采样问题。
  - 多方法交叉验证是官方优秀论文的高频特征。

## 横向观察

1. 官方优秀论文摘要都非常密集，几乎没有空泛背景段。
2. 每个问题都明确给出“方法 + 结果”，且常含具体数值。
3. A/B 更偏几何和工程物理，C/E 更偏数据分析，D 更偏生产计划和优化。
4. 优秀论文常组合多个算法，但组合不是堆砌，而是按“数据处理 -> 模型建立 -> 求解 -> 验证/优化”串起来。
5. 生成器需要支持题型识别后自动推荐模型链，而不是只推荐单个算法。
