## Reading Rule

When a new problem arrives:

1. identify the route family;
2. open this matrix to pick the positive sample and the comparison sample;
3. read the deep-reading anchor for the route;
4. read the risk gate before drafting;
5. then move into the paper-family matrix and generation loop.

## Why This Exists

The project already knows how to classify routes. This note makes the next step faster:

- which positive paper to imitate first;
- which comparison note to use as a contrast;
- which deep-reading note carries the route backbone;
- which risk note should guard against fake completion.
