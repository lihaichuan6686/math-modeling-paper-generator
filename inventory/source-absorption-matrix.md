# Source Absorption Matrix

Purpose: connect each raw source bucket to the structured notes that already absorb it and the next best conversion target.

This is a coverage-to-action ledger, not an archive.

## Bucket To Coverage Map

| Bucket | Already absorbed into | What is still thin | Next conversion target |
|---|---|---|---|
| CUMCM historical problems and official excellent papers | route notes, deep reads, official-paper paradigms, route-example matrix, archetype-section matrix, full-stack execution map | more same-problem comparisons and more route-specific anchor notes | additional comparison notes for recurring route clusters |

| Generated run artifacts and review evidence | run artifact index, quality evidence index, review finding index, finding-gate matrix, completion gate, review notes | more cross-run comparison and stronger PDF-layout evidence | an artifact comparison note or richer run review note |

## Update Rule

When a new raw source bucket is absorbed:

1. convert it into at least one structured note;
2. update the relevant route, section, or review bridge;
3. record only the coverage change here.
