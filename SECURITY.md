# Security and Responsible Use

This repository is private by default because it can produce complete math-modeling paper drafts.

## Intended Uses

- Research on human-written and AI-assisted math-modeling papers.
- Post-contest review and teaching.
- Reproducibility analysis.
- Paper quality checking and formatting assistance.

## Disallowed Uses

- Submitting generated work to an active contest in violation of contest rules.
- Hiding AI involvement where disclosure is required.
- Fabricating data, references, experiments, or authorship.
- Using the tool to bypass academic or competition integrity policies.

## Maintenance Policy

- Keep the repository private unless a careful public-release review is completed.
- Avoid adding one-click workflows that encourage misuse during active contests.
- Prefer quality-checking and reproducibility features over unattended final-paper generation.
- Document generated artifacts and model assumptions for auditability.
